# Cardápio
Cardápio de uma cafeteria fícticia chamada Cafezinho.

## Atividade
 Atividade cardápio elasNaTech:Criação de um cardápio simples usando html e css

 ## Estrutura
 - Abas
   - Café
   - Salgados
   - Sobremesas
 - Em todas a seções há:
   - Imagens do produto;
   - Valores

  ## Ferramentas utilizadas:
  - HTML
  - CSS
